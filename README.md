# GetAJob

AI-assisted job search skill for OpenClaw and Claude. GetAJob helps you speed up your job search, build your dream companies list, hunt for existing listings, and draft outreach.

## Project Structure

```
getajob-site/
├── public/
│   ├── index.html
│   ├── openclaw/
│   │   ├── install.sh
│   │   ├── getajob-openclaw.zip
│   │   └── skill/
│   │       └── SKILL.md
│   └── claude/
│       ├── install.sh
│       ├── getajob-claude.zip
│       └── skill/
│           ├── SKILL.md
│           ├── scripts/
│           └── references/
└── README.md
```

## Install

OpenClaw:
```bash
curl -fsSL https://getajob.bitpixi.com/openclaw/install.sh | bash
```

Claude:
```bash
curl -fsSL https://getajob.bitpixi.com/claude/install.sh | bash
```
