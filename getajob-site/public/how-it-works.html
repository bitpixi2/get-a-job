How It Works: GetAJob helps to speed up your job search, build your dream companies list, hunt for existing listings in your preferences, and drafts your outreach. Good luck!

OpenClaw Install Layout

OpenClaw installs the skill and links it into your OpenClaw skills folder.

~/getajob-openclaw/ └── SKILL.md ~/clawd/skills/getajob -> ~/getajob-openclaw

Claude Install Layout

The Claude installer creates a dedicated folder with scripts and references.

~/getajob-claude/ ├── SKILL.md ├── scripts/ │ ├── init_workspace.py │ ├── add_listing.py │ └── update_dream_company.py └── references/ ├── search-strategies.md ├── csv-schemas.md ├── application-materials.md └── summary-templates.md

What Happens Next

Once installed, you load the skill in Claude or restart the OpenClaw gateway. The assistant can then read the local files, run the helper scripts, and keep your job search data organized.

Tip: You can inspect all files before running the install command. It's open source.
